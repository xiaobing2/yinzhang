# 在线印章生成器（Edge Seal）

一个部署即用的在线印章生成器：输入 2-4 个字/字符，浏览器端实时生成“篆刻风格”的方形印章，并支持一键下载 PNG。项目同时提供一个边缘函数接口，用于展示 Pages 的边缘计算能力。

**本项目由阿里云ESA提供加速、计算和保护**  
![本项目由阿里云ESA提供加速、计算和保护](./assets/esa.png)

## 作品介绍（评审更关心的三点）

- **实用性（应用价值）**：用于电子合同/文件预览盖章、个人签名章、作品水印章等场景；无需注册、无需后端，打开即用。
- **创意性（创意卓越）**：用“红底白边+阴刻字+随机纹理噪声”模拟传统印章质感；生成过程可视化，适合传播分享。
- **技术深度（技术探索）**：
  - 使用 **ESA Pages** 托管静态站点（`index.html` + `styles.css` + `script.js`）。
  - 使用 **边缘函数**（`functions/timestamp.js`）提供轻量 API（返回服务器时间 ISO 字符串），可用于演示：边缘计算、低延迟请求、后续扩展（如：生成记录、限流、缓存等）。

## 功能特性

- 输入 2-4 字/字符，实时生成方形印章预览
- Canvas 生成图像，支持 **下载 PNG**
- 简单“做旧/刻痕”纹理：通过像素级随机噪声模拟雕刻质感
- 边缘函数示例：返回当前时间 JSON

这是一个静态站点项目，直接用任意静态服务器启动即可：

- VS Code Live Server
- 或使用任意 http server（例如 `npx serve` / `python -m http.server`）
